Flex-box is single dimentional meaning it can move either vertically or horizontally; but not both at once.
Grid Layout on the hand is two-dimentional as it can move in both directions at the same time.

Bootstrap is less flexible than Tailwind. It consumes larger file size than Tailwind due to its required files.
Tailwind is becoming more famous over the years because of its flexibility and more utility classes than Bootstrap.
But downside to Tailwind is that code is not clean make the users hard to read.

The CSS box model is essentially a box that wraps around every HTML element.
It consists of: margins, borders, padding, and the actual content or texts.

Semantic tag is the use of HTML markup to reinforce the information in web pages and web applications.
div and span tags are non-semantic.
article, header, section, main, form, footer are Sementic tags.
